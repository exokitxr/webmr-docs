## Launch a URL with Exokit Engine

1. Install Exokit.

2. Navigate to /exokit directory.

3. `./exokit.cmd yourURL`

## Launch Exokit Browser

1. Install Exokit

2. Navigate to /exokit directory.

3. `./exokit.cmd -h`


## Flags

**Version of Exokit Engine**
- `-v` 

**Launch Exokit Browser**
- `-h` 

**Reality Tab**
- `-t`

**WebGL**
- `-w` 

**WebXR**
- `-x` 

**Uncap FPS**
- `-u`

**Performance logging to Console**
- `-p` or `-perf` 

**Size**
- `-s`

**Frame**
- `-f`

**minimalFrame**
- `-m`

**Quit**
- `-q`

**Blit**
- `-b`

**Image**
- `-i`

**Require**
- `-r`
