## Requirements

1. Install Exokit.

2. Navigate to /exokit directory.

## Launch a URL with Exokit Engine

`./exokit.cmd yourURL`

## Launch a local file with Exokit Engine

`./exokit.cmd /your/File/Path.html`

## Launch Exokit Browser

`./exokit.cmd -h`
