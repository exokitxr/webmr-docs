Command Line Flags

An example of how to use these flags:

./exokit.cmd -hup

-hup will launch Exokit Browser in a uncapped FPS mode with console performance logging.

Short-Form FlagLong-Form FlagDescription
-v--versionVersion of Exokit Engine
-h--homeLaunch Exokit Browser
-t--tabReality Tab
-w--webglWebGL
-x--xrWebXR
-u--uncappedUncap FPS
-p--performancePerformance logging to Console
-s--sizeSize
-f--frameFrame
-m--minimalFrameminimalFrame
-q--quitQuit
-b--blitBlit
-i--imageImage
-r--requireRequire
